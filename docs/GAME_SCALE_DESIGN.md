# Game Scale Design System
# 游戏尺度设计规范

> 本文档是项目的核心设计参考，定义了游戏中所有实体、空间和时间相关的数值标准。

---

## 设计目标

| 指标 | 目标值 |
|---|---|
| 游戏时长 | 30分钟 |
| Boss数量 | 5个 |
| 地图数量 | 4张大地图 |
| 风格 | 2D版Warframe快节奏战斗 |

---

## 第一部分: 基础尺度单位

### 逻辑分辨率
```
640 × 360 像素 (16:9)
```

### 黄金比例定义

| 概念 | 定义 | 像素值 |
|---|---|---|
| **1 单位 (Unit)** | 基础参考单位 | 8 px |
| **1 角色宽度** | 玩家/标准敌人 | 3-4 Units = **24-32 px** |
| **1 格子 (Tile)** | 地图最小构成 | 4 Units = **32 px** |
| **1 房间** | 战斗区域 | 20-30 Tiles = **640-960 px** |
| **闪避距离** | 1次闪避移动 | 约 2 角色宽度 = **48-64 px** |
| **武器近程** | 霰弹枪/近战范围 | 3-4 角色宽度 = **80-120 px** |
| **武器中程** | 步枪有效射程 | 10-15 角色宽度 = **300-450 px** |
| **武器远程** | 狙击枪有效范围 | 20-25 角色宽度 = **600-800 px** |

---

## 第二部分: 实体尺寸

### 玩家
| 属性 | 设计值 | 屏幕占比 |
|---|---|---|
| Size | **28 px** | 4.4% |
| Hitbox | **24 px** | 碰撞判定略小于视觉 |

### 敌人等级

| 类型 | 尺寸 | 相对玩家 | 例子 |
|---|---|---|---|
| **小型** | **18 px** | 0.65× | Bat, 小虫 |
| **标准** | **24 px** | 0.85× | Skeleton, Charger |
| **中型** | **32 px** | 1.15× | Plant, Gunner |
| **重型** | **40 px** | 1.4× | Brute, Heavy |
| **精英** | **48 px** | 1.7× | Elite任意 |
| **Boss** | **64 px** | 2.3× | Treant, 各Boss |
| **大Boss** | **80 px** | 2.9× | 最终Boss |

---

## 第三部分: 移动速度

### 速度参考 (像素/秒)

| 实体 | 速度 | 设计意图 |
|---|---|---|
| **玩家行走** | **140** | 基础移动感流畅 |
| **玩家冲刺** | **210** | 战术走位 |
| **玩家闪避** | **500** | 短距快速 |
| **小型敌人** | **100** | 必须比玩家慢 |
| **标准敌人** | **60** | 容易风筝 |
| **冲锋敌人** | **500** | 需要闪避 |
| **Boss** | **50** | 有压迫感但可控 |

### 速度规则
```
玩家基础速度 > 所有常规敌人
闪避速度 > 敌人冲锋速度
冲刺速度 = 小型敌人速度 × 2
```

## 第四部分: 投射物参数

### 玩家子弹

| 武器类型 | 速度 (px/s) | 大小 (px) | 设计意图 |
|---|---|---|---|
| **步枪** | 1200 | 8 | 接近瞬发 |
| **手枪** | 1100 | 6 | 快速响应 |
| **霰弹枪** | 800 | 5 | 可见弹幕 |
| **狙击枪** | 1800 | 12 | 穿透感 |
| **弓箭** | 900 | 10 | 可见轨迹 |
| **能量武器** | 600-800 | 4-14 | 视觉丰富 |

### 敌人子弹

| 类型 | 速度 (px/s) | 大小 (px) | 设计意图 |
|---|---|---|---|
| **普通** | 180 | 10 | 容易躲避 |
| **快速** | 280 | 8 | 需要注意 |
| **Boss** | 220-300 | 14 | 明显威胁 |

### 子弹规则
```
玩家子弹速度 >> 敌人子弹速度 (4-5倍)
敌人子弹必须大且可见
Boss弹幕应该可预判
```

## 第五部分: 战斗空间设计

### 区域类型

| 区域类型 | 尺寸 (Tiles) | 尺寸 (Pixels) | 相对屏幕 | 用途 |
|---|---|---|---|---|
| **走廊** | 4×20 | 128×640 | 0.5× | 过渡，追逐 |
| **小战斗区** | 15×15 | 480×480 | 0.9× | 小规模遭遇 |
| **标准战斗区** | 20×20 | 640×640 | 1.2× | 主要战斗 |
| **大战斗区** | 25×25 | 800×800 | 1.5× | 精英战/多波次 |
| **Boss房** | 30×30 | 960×960 | 1.8× | Boss战 |
| **大Boss房** | 40×40 | 1280×1280 | 2.5× | 最终Boss |

### 战斗空间规则
```
战斗区最小 = 玩家武器射程 × 1.5
战斗区最大 = 屏幕对角线 × 2
玩家应能看到大部分敌人
必须有足够空间闪避
```

## 第六部分: 地图结构

### 宏观结构 (30分钟游戏)

```
地图1 (8分钟) → 地图2 (7分钟) → 地图3 (7分钟) → 地图4 (8分钟)
    ↓              ↓              ↓              ↓
  Boss1          Boss2          Boss3+4        Boss5(Final)
```

### 每张地图结构

```
┌─────────────────────────────────────┐
│ 入口区 → 战斗区1 → 过渡走廊        │
│              ↓                      │
│         战斗区2 ← 秘密区域          │
│              ↓                      │
│    分支A ←──╋──→ 分支B              │
│              ↓                      │
│         Mini-Boss区                 │
│              ↓                      │
│    Boss准备区 → Boss房              │
└─────────────────────────────────────┘
```

### 单张地图尺寸

| 组件 | 数量 | 尺寸 | 总面积 |
|---|---|---|---|
| 走廊 | 4-6条 | 128×640 | ~400k px² |
| 小战斗区 | 3-4个 | 480×480 | ~700k px² |
| 标准战斗区 | 2-3个 | 640×640 | ~1.2M px² |
| 大战斗区 | 1-2个 | 800×800 | ~1M px² |
| Boss房 | 1个 | 960×960 | ~900k px² |
| **总计** | | | **~4M px²** |

相当于 **约 64×64 Tiles** 或 **2048×2048 像素** 的正方形区域

## 第七部分: 时间分配

### 单张地图流程 (约7-8分钟)

| 阶段 | 时间 | 内容 |
|---|---|---|
| 探索+战斗 | 3-4分钟 | 清理小战斗区，获取资源 |
| 分支选择 | 1分钟 | 选择路径，可选挑战 |
| Mini-Boss | 1分钟 | 中等难度精英战 |
| Boss准备 | 30秒 | 最后补给 |
| Boss战 | 2-3分钟 | 核心战斗 |

## 第九部分: 战斗节奏

### 击杀时间 (TTK)

| 敌人类型 | 步枪 | 霰弹枪 | 狙击枪 |
|---|---|---|---|
| 小型 | 2-3发 | 1发 | 1发 |
| 标准 | 4-6发 | 1-2发 | 1发 |
| 中型 | 8-10发 | 2-3发 | 1-2发 |
| 重型 | 15-20发 | 4-5发 | 2-3发 |
| 精英 | 25-30发 | 6-8发 | 4-5发 |
| Boss | 60-100发 | 20-30发 | 15-20发 |

### 节奏感
```
每秒击杀 = 0.5-1.5 敌人 (普通战斗)
每波敌人 = 8-15只 (小战斗区)
每波敌人 = 20-30只 (大战斗区)
Boss战持续 = 2-3分钟
```
