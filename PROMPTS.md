# Asset Prompt Templates / 素材提示词模板

Use these ready-made prompts with your image generator. Replace bracketed variables. Transparent backgrounds where applicable.

## Character Move Sprite Sheet (4 frames horizontal strip) / 角色移动图集（4 帧横向条带）

**Template / 模板**
```text
EN: Pixel art character move cycle sprite sheet, 4 frames horizontal strip, each frame [WxH], high angle view, facing right, character: [character/look/theme], light outline, simple shading, consistent silhouette, transparent background.
ZH: 像素风角色移动循环图集，4 帧横向条带，单帧 [宽x高]，高视角，面向右侧，角色：[角色/造型/主题]，轻描边，简单明暗，轮廓一致，透明背景。
```

**Example / 示例**
```text
EN: Pixel art character move cycle sprite sheet, 4 frames horizontal strip, each frame 32x32, high angle view, facing right, character: a green-cloaked ranger with a shortbow and gold trim, light outline, simple shading, consistent silhouette, transparent background.
ZH: 像素风角色移动循环图集，4 帧横向条带，单帧 32x32，高视角，面向右侧，角色：披绿斗篷的游侠，手持短弓金色装饰，轻描边，简单明暗，轮廓一致，透明背景。
```

## Ground Tile (64x64 seamless) / 地面平铺纹理（64x64 无缝）

**Template / 模板**
```text
EN: Seamless pixel art [surface] tile, 64x64 pixels, low-contrast noise, subtle color variation, occasional tiny details, no large objects, for top-down background, edges should tile seamlessly.
ZH: 像素风无缝 [材质] 平铺块，64x64 像素，低对比噪声，细微色差，少量小细节，不要大物体，用于顶视角背景，边缘需无缝衔接。
```

**Example (Grass) / 示例（草地）**
```text
EN: Seamless pixel art grass tile, 64x64 pixels, low-contrast noise, subtle color variation, occasional small blades or clovers, no stones or large objects, for top-down background, edges should tile seamlessly.
ZH: 像素风无缝草地平铺块，64x64 像素，低对比噪声，细微色差，偶有小草叶或三叶草，没有石头或大物体，用于顶视角背景，边缘需无缝衔接。
```
